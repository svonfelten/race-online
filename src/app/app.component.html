Race

Race againsts other competitors and try to get to all targets with the minimum amount of moves

Each game happens in "frames" or "moves". Each car will be asked to return a Vector at the end of the frame. This 2D vecor can only be changed by 1 each for example you can go from { x: 0, y: 1} -> { x: 1, y: 2} but not { x: 2, y: -1}.

So after each car has setted his move the vectors will be applied to the current positions. The main goal is to get as much speed as possible without going out of the map and without cheating.
